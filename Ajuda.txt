Como Colaborar
Para colaborar neste projeto, siga as instruções abaixo:

1. Clonar o Repositório
Antes de começar a trabalhar no projeto, clone o repositório para o seu computador local usando o seguinte comando:
git clone https://github.com/RafaelyOz/Florentine

2. Criar uma Nova Branch para o Frontend
Antes de começar a trabalhar nas alterações do frontend, crie uma nova branch para suas alterações usando o seguinte comando:
git checkout -b front-end

3. Desenvolver e Atualizar o Frontend
Faça suas alterações no código-fonte do frontend conforme necessário. Certifique-se de fazer commits regulares para registrar seu progresso:
git add .
git commit -m "Descrição concisa das alterações"

4. Enviar Atualizações para a Branch do Frontend
Após fazer suas alterações e commits, envie suas atualizações para a branch do frontend usando o seguinte comando:
git push origin front-end
